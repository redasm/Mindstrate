# Mindstrate

Choose your language:

- [中文](README.zh-CN.md)
- [English](README.en.md)

Mindstrate is a memory and context substrate for AI coding agents. It captures reusable engineering knowledge, session continuity, external repository signals, and project graph context, then exposes them through MCP, CLI, Team Server, Web UI, and Obsidian projections.
