# Mindstrate

Choose your language:

- [中文](README.zh-CN.md)
- [English](README.en.md)

Mindstrate is an Evolvable Context Substrate (ECS) for AI coding agents. It treats memory as a metabolizing compression lineage: raw work episodes become snapshots, summaries, patterns, skills, and rules, while stale or low-value context is pruned. It exposes this governed working context through MCP, CLI, Team Server, Web UI, and Obsidian projections.
