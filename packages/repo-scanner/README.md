# @mindstrate/repo-scanner

`@mindstrate/repo-scanner` is Mindstrate's external data collection tool.

It owns Git, Perforce, hooks, daemon polling, cursors, failed item retry, and custom source adapters. The core framework only accepts standardized Mindstrate input; it does not read Git/P4/watch sources directly.
